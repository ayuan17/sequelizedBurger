# Sequelized Burger Project
https://sql-project.herokuapp.com/ <br/>
Project using SQL and Sequelize (A promise based object relation mapper for Node.js. Supports PostGRE SQL, MySQL, SQLite, and MSSQL) <br/>
You create a new burger in the SQL datebase, it will store the new burger in the database, and you can devour (delete) the burger from the database and it will display all devoured burgers
